A cosa serve bootstrap?

Bootstrap è una libreria di componenti front end e un toolkit open source. Viene usato per creare pagine web reattive e mobile-first con HTML, CSS e JavaScript.

Cosa devi importare per creare una pagina web con bootstrap

Per utilizzare il codice compilato di Bootstrap Italia nel proprio progetto, è sufficiente scaricare ed includere nella pagina HTML alcuni file, tra cui: il foglio di stile contenuto nel file CSS. le funzionalità dei vari componenti contenute nel bundle JavaScript. i font.

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Una rete di distribuzione di contenuti (CDN) è una rete di server interconnessi che velocizza il caricamento di pagine Web per applicazioni ad alto contenuto di dati. CDN può significare sia rete di consegna di contenuti che rete di distribuzione di contenuti.

A cosa serve questo attributo del tag meta?

indica al browser di adattare la larghezza della pagina alla larghezza del dispositivo e di impostare lo zoom iniziale a 1.

Che differenza c'è tra Container e Container Fluid

Container fisso: ha una larghezza fissa e viene usato per creare layout con margini fissi. Container fluido: ha una larghezza flessibile e si adatta alla larghezza della finestra del browser.
