Colour classes

Classi di colori Bootstrap

.text-muted
- la classe text-muted crea stile di testo grigio chiaro a un elemento HTML
.text-primary
- la classe text-primary è una classe di testo che definisce il colore del testo di un elemento HTML. In particolare, text-primary è progettata per assegnare al testo un colore che corrisponde al colore primario definito nel tema di Bootstrap
.text-success
- La classe text-success di Bootstrap è una classe di testo che viene utilizzata per cambiare il colore del testo in verde per indicare successo o conferma
.text-info
- la classe text-info viene utilizzata per applicare uno stile di colore specifico al testo all'interno di un elemento HTML. Questa classe è parte del sistema di colori di Bootstrap ed è utilizzata per rendere il testo di un elemento visivamente evidenziato con un colore blu scuro
.text-warning
- la classe text-warning è utilizzata per applicare uno stile di testo con un colore di avvertimento (solitamente un colore giallo) a un elemento HTML
.text-danger
- La classe text-danger in Bootstrap è una classe di utility utilizzata per applicare uno stile di colore specifico al testo all'interno di un elemento HTML. In particolare, questa classe assegna al testo il colore rosso
.text-secondary
- la classe text-secondary è utilizzata per applicare uno stile di colore grigio chiaro ad un elemento html
.text-white
- la classe text-white è utilizzata per applicare uno stile di colore bianco ad un elemento html
.text-dark
- la classe text-dark è utilizzata per applicare uno stile di colore grigio scuro ad un elemento html
.text-body
- la classe text è utilizzata per applicare uno stile di colore nero ad un elemento html
.text-light
- la classe text-secondary è utilizzata per applicare uno stile di colore grigio chiaro (su sfondi bianchi) ad un elemento html
